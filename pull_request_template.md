- 1. 개발 과정에서 SQL의 data type 자동 변환 처리가 생각보다 많은 데이터 처리 소요가 있겠다는 생각이 들었으며, db의 내용과 서버에서 접근하는 데이터 양이 늘어날 수록 이를 사전에 조절하는 방법을 더 구체화하여 적용해야 겠다는 생각이 들엇습니다.
